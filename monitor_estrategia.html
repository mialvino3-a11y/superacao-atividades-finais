Como os dados foram calculados — Nota Metodologica
De onde vem os 51.464 registros?
O SIGMA gera o arquivo Relatorio_PDF_Estrategia_22062026_FGV.xlsx com todas as atividades pactuadas nos PDFs. O arquivo bruto tinha ~56 mil linhas. Como o mesmo encaminhamento pode aparecer varias vezes quando o agente atualiza o status, mantivemos apenas o registro mais recente de cada atividade por familia e pessoa/representante. Resultado: 51.464 atividades unicas.
Unidade de contagem: familia + pessoa/representante + politica + objetivo + acao + atividade. Assim, uma mesma familia pode aparecer mais de uma vez se tiver mais de uma pessoa/representante ou mais de uma atividade/politica. Nao e uma contagem de familias unicas.
Status no topo: Nao iniciadas = status literal "Nao iniciada"; Em andamento = status literal "Em andamento"; Concluidas = status literal "Concluida"; Sem status preenchido = campo "Status da Atividade" em branco; demais status com execucao/intercorrencia somam 2.052 registros.
Como e calculado o % Nao Iniciada?
Para cada recorte (politica, bloco ou municipio), contamos quantas atividades tem status "Nao Iniciada" e dividimos pelo total daquele recorte. O grafico de barras mostra essa distribuicao empilhada para 100% das atividades de cada politica, ordenado da mais critica (topo) para a mais avancada (base).
Formula: % NI = (qtd. com status "Nao Iniciada") / (total de atividades do recorte) x 100
Como e calculada a Evolucao Mensal?
Cada atividade tem uma data de registro no sistema (quando o agente inseriu ou atualizou aquele encaminhamento). Agrupamos por esse mes e calculamos o % NI de cada grupo. Atencao: junho concentra 33.417 registros (65% do total) porque muitos agentes lancaram dados retroativos nesse mes — a curva de junho nao reflete necessariamente piora operacional, mas volume de lancamento tardio.
Formula por mes: % NI do mes = (atividades registradas naquele mes com status NI) / (total registradas naquele mes) x 100
O que define um "PDF Sem Execucao" e como calcular dias?
Um PDF e considerado sem execucao quando 100% das suas atividades estao com status "Nao Iniciada". Os dias sao calculados entre a data de pactuacao do PDF (coluna do SIGMA) e a data da extracao (22/06/2026). PDFs com 60+ dias sem nenhuma atividade iniciada sao classificados como criticos.
Limite: 0–29 dias = Atencao · 30–59 dias = Alerta · 60+ dias = Critico
Os numeros refletem a realidade operacional?
Os dados dependem inteiramente do que o agente registrou no SIGMA. Se um encaminhamento foi iniciado mas nao atualizado no sistema, ele aparece aqui como "Nao Iniciado". O painel mede a execucao registrada, nao necessariamente a execucao real. Lacunas de registro podem subestimar o avanco.
