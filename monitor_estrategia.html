O que e Execucao dos PDFs
O PDF (Plano de Desenvolvimento Familiar) reune todos os encaminhamentos previstos para uma familia. Um PDF esta "Sem execucao" quando 100% das atividades previstas estao com status "Nao Iniciada" — a familia nao foi efetivamente encaminhada a nenhuma politica. O numero de dias indica ha quanto tempo o PDF foi emitido sem que nenhuma atividade tivesse inicio. PDFs parados ha mais de 60 dias sao criticos.

Nota Metodologica — Como os dados foram calculados
Fonte e preparacao dos dados

Os dados partem do arquivo Relatorio_PDF_Estrategia_22062026_FGV.xlsx, gerado pelo sistema SIGMA em 22/06/2026. Cada linha do arquivo representa uma atividade prevista no Plano de Desenvolvimento Familiar (PDF) de uma familia — ou seja, um encaminhamento a uma politica publica estadual (ex.: Banco do Povo, creche, UBS).

O arquivo bruto contem registros duplicados porque o sistema registra cada atualizacao de status da mesma atividade como uma nova linha. Para evitar dupla contagem, aplicamos deduplicacao: para cada combinacao unica de (familia + politica publica + objetivo + acao + atividade), mantemos apenas o registro mais recente (pela data de registro). O resultado final e de 51.464 atividades unicas.

Aba "Por Politica Publica" — grafico de barras empilhadas

Para cada politica publica, calculamos a distribuicao percentual de status do total de atividades daquela politica:

% de cada status = (qtd. de atividades com aquele status na politica) / (total de atividades da politica) × 100

O grafico ordena as politicas da que tem maior % de Nao Iniciada (topo, mais critica) ate a de menor (base, mais avancada). Os filtros de Bloco e Municipio recalculam o mesmo grafico apenas para o subconjunto selecionado, permitindo comparar a demanda e o status por regiao do programa.

Aba "Evolucao Mensal" — grafico de linhas e mapa de calor

Cada atividade tem uma data de registro (coluna "Data de Registro da Atividade" no SIGMA). Essa data indica quando o agente inseriu ou atualizou o registro no sistema — nao necessariamente quando a atividade ocorreu ou foi encaminhada.

Para cada combinacao de mes x bloco (ou mes x politica no mapa de calor), calculamos:

% Nao Iniciada no mes = (atividades registradas naquele mes com status "Nao Iniciada") / (total de atividades registradas naquele mes) × 100

Limitacao importante: junho/2026 concentra 33.417 registros (65% do total), pois muitos agentes inseriram dados historicos no sistema nesse periodo. Por isso, a comparacao entre meses reflete parcialmente o ritmo de lancamento de dados, e nao apenas o avanco operacional real. Recomenda-se interpretar os meses iniciais (jan–mar) com cautela por causa do volume menor de registros.

Aba "Execucao dos PDFs" — familias sem execucao

Um PDF (Plano de Desenvolvimento Familiar) e considerado "Sem execucao" quando 100% das atividades registradas para aquela familia estao com status "Nao Iniciada" — ou seja, nenhum encaminhamento teve qualquer avanco.

Os indicadores sao calculados da seguinte forma:

  • Familias com PDF: total de codigos de familia com ao menos uma atividade registrada.
  • Sem execucao: familias em que todas as atividades estao "Nao Iniciada".
  • Dias sem execucao: diferenca entre 22/06/2026 (data da extracao) e a data de pactuacao do PDF (quando o PDF foi elaborado com a familia). Indica ha quantos dias aquela familia espera sem nenhum encaminhamento iniciado.
  • Criticos (60+ dias): familias com PDF parado ha mais de dois meses.

A classificacao por categoria usa o percentual de atividades Nao Iniciadas: Com bom andamento (<30% NI) · Em andamento parcial (30–69%) · Pouco avancado (70–99%) · Sem execucao (100%).

Sobre os status das atividades

Os status refletem o que o agente registrou no SIGMA. Os possiveis valores sao: Nao iniciada (sem avanco), Em andamento, Concluida, Atrasada, Desistiu/Perdeu o interesse, Nao realizada e Nao se aplica. A qualidade das informacoes depende da atualizacao sistematica dos registros pelos agentes no sistema.
